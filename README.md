# Igor Pukalski — Creative Resume

Lightweight single-page resume site for creative and multimedia work.

## Stack

- Next.js 15
- React 19
- Tailwind CSS v4
- Light / dark theme toggle

## Development

```bash
pnpm install
pnpm dev
```

## Deploy

Deploy the `creative-resume` branch on Vercel (Hobby plan). No env vars required for a basic deploy.

Optional: set `NEXT_PUBLIC_SITE_URL` for canonical metadata.

## Branch

This branch is separate from `main` (dev portfolio auto-ship). Content is synced from `resume collection/` into `src/data/`.
